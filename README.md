Happycome - ミニマル企業サイト

**シンプルで洗練されたミニマルデザインの企業LP**（練習サンプル）

書籍『動くWebデザインアイディア帳』を参考に作成した、コンサルティング会社のコーポレートサイトです。  
スクロールに連動した背景の伸びアニメーション、スクロールプログレスバー、ニュースティッカー、アコーディオン、スマホ用モーダルメニューなど、細部までこだわった演出が満載です。

## デモ（ライブプレビュー）
https://minimal-company-htmlcssjs-practice.vercel.app/


## 主な実装機能
- **スクロールプログレスバー**（scrollgress.js） - ページ上部にオレンジの進捗バー
- **背景が伸びて文字が出現**（bgLRextend / bgappear） - セクションごとに左から右へ伸びるアニメーション
- **左から回転登場**（flipLeftTrigger） - Visionエリアの画像とテキスト
- **下からふわっと登場**（fadeUpTrigger） - Aboutテーブル、FAQ、Contactフォーム
- **ニュースティッカー**（bxSlider） - Topicsエリアで縦スライド
- **アコーディオン**（FAQ） - クリックで開閉、最初の一つだけ初期オープン
- **スマホ用モーダルメニュー** - 右上のボタンで下から出現
- **ページトップボタン** - スクロール200px超えたら右から出現
- **現在地ハイライト** - 上部ナビがスクロール位置に応じてアクティブ表示（文字がオレンジに）
- **スムーススクロール** - ナビクリックでヘッダー分を考慮した位置へ移動

## 使用技術・ライブラリ
- HTML5 / CSS3
- jQuery 3.4.1
- [scrollgress.js](https://github.com/yuheiy/scrollgress) - スクロールプログレスバー
- [bxSlider](https://bxslider.com/) - ニュースティッカー
- Google Fonts（Lato）
